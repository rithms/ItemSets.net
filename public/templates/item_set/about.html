About and F.A.Q

About

This web app was created by rithms as an entry to the second Riot Games API Challenge. It was submitted as an entry to the third contest category - 'Develop a piece of software that creates value a benefit for players utilizing the recent release of the Item Set documentation.'

This project is open source. The source code and documentation can be viewed on GitHub.

What are Featured Item Sets?

Featured Item Sets are item sets generated from Featured Games. Featured ranked solo games are typically high elo games, so you will see a lot of item sets generated from your favorite pro player's builds

How are Item Sets Generated?

The Item Sets found on this site are generated from build orders found in high elo games. Once a featured game ends, the build order of each player is analyzed. Items are placed into 2 categories, 'Starting Items', and 'Final Build'. From there, items from both categories are taken and added to another three categories, 'Offensive Items', 'Defensive Items', and 'Consumables'. Currently, an item is placed into the 'Offensive Items' category if it gives spell damage, CDR, attack damage, critical strike, attack speed, or lifesteal. Secondly, an item is placed into the 'Defensive Items' category if it gives health, armor, or magic resist. Lastly, an item is placed into the 'Consumables' category if it gives vision, or is a consumable item.

Please note the method for which item sets are generated could change in the future, as I work to improve this project.
