Item Sets - Tutorial

What are Item Sets?

An item set is a set of items that can be displayed in the in-game shop of League of Legends. An item set can consist of multiple blocks, used to categorize items in the same set.

As an example, below you can see an item set named 'Best Item Set NA', consisting of multiple blocks named 'Starting Items', 'Offensive Items', 'Defensive Items', 'Final Build', and 'Consumables'.

Item sets allow you to associate items with specific champions and maps so you don't have to waste your precious time playing hide-and-seek with the in-game store.

Custom Sets vs. Client Sets

Client created sets are item sets that are created using the game client. These item sets are associated with your account, and are loaded in when you start a game.

Custom created sets are item sets created outside the game client. These item sets are stored locally on your computer, and aren't associated with your account. This means custom created item sets won't follow you if you switch computers like client created item sets do. Custom created item sets won't be visible on the client under the Item Sets tab, but will still be loaded in game.

On this site, you will only find custom created item sets. However, you are free to create a client created item set yourself, inspired by any of the generated sets you find on this site.

File Location

To load item sets into the game, you will need to save the JSON data provided to you on this site into a JSON file (i.e. Item_Set.json), and place the file in it's respective directory.

You can place the file in the global directory to make the item set available for all champions, or you can place the file in a champion specific directory using the champion's key name.

Windows

All Champion Directory: League of Legends\Config\Global\Recommended\

Champion Specific: League of Legends\Config\Champions\{championKey}\Recommended\

Mac

All Champion Directory: League of Legends.app/Contents/LoL/Config/Global/Recommended/

Champion Specific: League of League of Legends.app/Contents/LoL/Config/Champions/{championKey}/Recommended/


For more information about Item Sets, refer to the Riot Games API Item Set Documentation.
